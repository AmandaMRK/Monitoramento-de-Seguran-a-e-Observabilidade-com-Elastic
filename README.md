# Monitoramento de Segurança e Observabilidade com Elastic

Este projeto demonstra a implementação de um ambiente de monitoramento de segurança utilizando a stack Elastic. O objetivo é coletar e analisar logs de sistema para identificar eventos de autenticação e garantir a visibilidade dos processos.

## O que foi feito:
* **Instalação e Configuração**: Implementação do Elastic Agent para coleta de dados de log do sistema operacional.
* **Monitoramento**: Configuração de filtros no Kibana (KQL) para isolar eventos de autenticação (`system.auth`).
* **Visualização**: Análise de logs em tempo real para observabilidade e detecção de padrões.

## Tecnologias Utilizadas:
* Elastic Stack (Kibana, Elastic Agent, Elastic Cloud)
* Linux (Ubuntu)
* Git/GitHub para versionamento de código
